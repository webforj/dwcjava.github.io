CLIENT COMPONENT

This section outlines styling information for the <dwc-numberfield-spinner> component. This component is client side only - it cannot be instantiated on its own via the API, but may make up part of API components.

Reflected Attributes​

The reflected attributes of a component will be shown as attributes in the rendered HTML element for the component in the DOM. This means that styling can be applied using these attributes.
